# 나의 업로드 저장소

어디서든 파일을 업로드하고, 다른 곳에서 다시 다운로드하기 위한 개인용 파일 업로드 사이트입니다.

## 기능

- 이름 + 비밀번호를 직접 정해서 그 자리에서 계정 생성 (회원가입 절차 없음, 처음 쓰는 이름이면 자동 등록)
- 등록된 이름을 맞는 비밀번호로 입력하면 그 계정의 파일 목록 조회 (계정마다 파일 공간 분리)
- 파일 업로드 시 설명(description) 함께 저장
- Vercel Blob에 파일 저장
- 저장된 파일 목록 조회
- 다운로드
- 삭제
- 관리자 페이지(`/admin`)에서 지금까지 만들어진 계정/비밀번호 목록 조회 (비밀번호 잊었을 때 확인용)

## 로컬 실행

```bash
npm install
cp .env.example .env.local
```

`.env.local` 파일을 열어서 관리자 비밀번호를 바꿉니다.

```env
ADMIN_PASSWORD=관리자비밀번호
```

이름/비밀번호는 사용자가 사이트에서 직접 정하는 것이라 여기서 미리 설정할 필요 없습니다.

실행:

```bash
npm run dev
```

브라우저에서 접속:

```txt
http://localhost:3000
```

## Vercel 배포

1. 이 폴더를 GitHub 저장소에 업로드
2. Vercel에서 New Project → GitHub 저장소 Import
3. Vercel 프로젝트의 Storage 탭에서 Blob 생성
4. Blob Store를 이 프로젝트에 연결
5. Vercel 프로젝트 Settings → Environment Variables에 추가

```env
ADMIN_PASSWORD=관리자비밀번호
```

6. Deploy

## 관리자 페이지

`/admin`에서 관리자 비밀번호(`ADMIN_PASSWORD`)를 입력하면, 사이트에서 사람들이 직접 만든 계정(이름+비밀번호) 목록을 볼 수 있습니다.  
비밀번호를 잊었을 때 여기서 확인하면 됩니다.

## 주의

- 이 사이트는 **누구나 URL만 알면 새 이름+비밀번호로 가입해서 업로드**할 수 있습니다. 이름이 이미 등록돼 있으면 그 비밀번호를 알아야만 접근 가능하지만, 가입 자체를 막는 절차는 없습니다. 아는 사람끼리만 URL을 공유하는 용도로 쓰는 걸 권장합니다.
- Vercel Blob **private** 스토어에 저장하며, 목록 조회/업로드/다운로드/삭제 전부 이름+비밀번호가 필요합니다.
- 개인/지인용 소형 파일 보관 용도로만 쓰는 것을 권장합니다.

Vercel Function의 요청/응답 body 제한이 있으므로 4MB 이하 파일만 업로드하도록 제한했습니다.
